
{{alias}}( value )
    Returns the enumeration constant associated with a supported SGD
    classification learning rate scheduler value.

    Downstream consumers of this function should *not* rely on specific integer
    values (e.g., `BASIC == 0`). Instead, the function should be used in an
    opaque manner.

    Parameters
    ----------
    value: any
        Learning rate scheduler value.

    Returns
    -------
    out: integer|null
        Enumeration constant.

    Examples
    --------
    > var out = {{alias}}( 'basic' )
    <number>
    > out = {{alias}}( {{alias:@stdlib/ml/base/sgd-classification/learning-rate-str2enum}}( 'basic' ) )
    <number>
