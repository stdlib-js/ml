
{{alias}}( learningRate )
    Returns the enumeration constant associated with an SGD classification
    learning rate scheduler string.

    Downstream consumers of this function should *not* rely on specific integer
    values (e.g., `BASIC == 0`). Instead, the function should be used in an
    opaque manner.

    Parameters
    ----------
    learningRate: string
        Learning rate scheduler.

    Returns
    -------
    out: integer|null
        Enumeration constant.
