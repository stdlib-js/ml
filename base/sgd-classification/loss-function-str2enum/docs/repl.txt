
{{alias}}( lossFunction )
    Returns the enumeration constant associated with an SGD classification loss
    function string.

    Downstream consumers of this function should *not* rely on specific integer
    values (e.g., `EPSILON_INSENSITIVE == 0`). Instead, the function should be
    used in an opaque manner.

    Parameters
    ----------
    lossFunction: string
        Loss function.

    Returns
    -------
    out: integer|null
        Enumeration constant.

    Examples
